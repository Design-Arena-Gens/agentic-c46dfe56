# Plan de contenu – écrans d'étiquetage digital

Application Next.js livrant une stratégie social media 30 jours (3 posts/jour) pour Facebook, Instagram et TikTok autour du lancement d'une solution d'écrans d'étiquetage électronique en France.

## Scripts

- `npm run dev` : lancer l'environnement de développement
- `npm run build` : générer le build de production
- `npm run lint` : vérifier la qualité du code
- `npm run generate:excel` : regénérer le fichier Excel `public/plan-contenu-ecrans-digitales.xlsx`

## Fonctionnalités

- Synthèse exécutive des messages, objectifs et KPI
- Tableaux éditoriaux complets (hook, copywriting, concepts créatifs, CTA, témoignages, prompts IA)
- Export Excel multi-feuilles prêt pour diffusion ou ajustement
